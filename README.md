# new-sakura starter

さく楽予約サイトのトップ画面たたき台です。

## 使い方
1. このZIPを解凍
2. GitHub Desktop か VS Code で開く
3. `npm install`
4. `npm run dev`
5. `public/salon-bg.jpg` に背景画像を入れる
6. GitHub の `dev` ブランチへアップロード

## 今入っているもの
- トップ画面
- ログイン前 / 後 / 管理者表示の見本
- メニューとオプションのポップアップ
- ◎ / ✕ の空き表示サンプル
- 前日メール文面表示欄
